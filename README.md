# React Zentry UI Clone

A hands-on recreation of the [zentry.com](https://zentry.com) landing page, built as a learning project to explore scroll-driven animations, responsive layout, and modern frontend tooling.

Not a pixel-perfect copy. The goal was understanding the techniques, not matching every detail.

## Tech Stack

| Tool | Purpose |
|---|---|
| React 19 + TypeScript | UI and type safety |
| Tailwind CSS | Utility-first styling |
| GSAP + @gsap/react | Scroll and entrance animations |
| Vite 6 | Dev server and bundler |

## Sections

- **Navbar** — sticky nav with scroll-aware behavior
- **Hero** — full-screen animated intro
- **About** — scroll-triggered reveal
- **Features** — animated feature cards
- **Story** — narrative section with transitions
- **Contact** — call-to-action block
- **Footer** — site footer

## Getting Started

```bash
npm install
npm run dev
```

| Script | Description |
|---|---|
| `npm run dev` | Start local dev server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |
